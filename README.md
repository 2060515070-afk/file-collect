# FileCollect - 文件收集站

一个现代化的文件收集与管理平台。

## 功能

- ✅ 创建文件收集任务
- ✅ 限制文件类型（PDF/Word/Excel/PPT/图片/视频/压缩包等）
- ✅ 分享链接给他人上交文件
- ✅ 选择姓名 + 提交文件
- ✅ 管理后台查看谁交了/谁没交
- ✅ 全部交完后自动打包发送到指定邮箱
- ✅ 暗黑科技风格 UI
- ✅ 移动端适配

## 使用

```bash
pip install -r requirements.txt
python app.py
```

管理后台：http://localhost:5000/admin/login
默认密码：admin123
